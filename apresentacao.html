Integração

Como usar

Três formas de incluir o design system no seu projeto.

1 · CDN (mais rápido)

Importe direto no HTML via jsDelivr — sem build, sem npm.

<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/gh/
fredparreira/fred-parreira-design-system/
components/all.css">
2 · Local

Clone o repositório e referencie o arquivo.

# Clone
git clone https://github.com/
fredparreira/fred-parreira-design-system

# No HTML
<link rel="stylesheet"
  href="caminho/components/all.css">
3 · Só os tokens

Use apenas as variáveis CSS no seu próprio sistema.

/* No seu CSS */
@import url('tokens/tokens.css');

/* Depois use as variáveis */
color: var(--fp-primary);
font-family: var(--fp-font-display);

HTML boilerplate

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <!-- Fonte display -->
  <link href="https://fonts.googleapis.com/css2?family=Exo+2:wght@300;400;500;700;900" rel="stylesheet">

  <!-- Design System completo -->
  <link rel="stylesheet" href="components/all.css">
</head>
<body>

  <h1 class="fp-display-xl">Título em Exo 2</h1>
  <p class="fp-body">Corpo de texto.</p>
  <a href="#" class="fp-btn-cta">Bora começar!</a>

</body>
</html>
